# Social Media Post Generator

When creating social posts:

1. Ask which business line (Family Law, DOT, DNA, Mobile Testing)
2. Load the brand voice guide from `docs/brand-voices/truetest.md`
3. Ask for the topic or source content (blog post, news, service highlight, etc.)
4. Generate platform-specific versions:
   - **LinkedIn** — professional tone, no emojis unless requested, 1300 char max, 3-5 relevant hashtags
   - **X (Twitter)** — punchy and direct, 280 char max per tweet, can create threads with 2-4 tweets
   - **YouTube** — community post style, 500 char max, include call-to-action
5. Output each as a separate markdown block ready to copy
6. Save output to `content/social/YYYY-MM-DD/` with filenames: `linkedin.md`, `x.md`, `youtube.md`

## Brand Voice Quick Reference

- **Family Law**: Expert consultant → attorneys/judges. Sharp, technical, peer-level. Frame around case outcomes.
- **DOT**: Straight-talking → fleet managers. Lead with price & compliance. Plain language.
- **DNA**: Reassuring → individuals. Affordable, no judgment, accessible.
- **Mobile**: Problem-solver → HR/attorneys. "We come to you. No excuses."

## Universal Rules
- Independent, family-run since 2012 — use as differentiator
- Always position as expert, never commodity
- No emojis on LinkedIn unless requested
- Fact-check all detection windows and technical claims
- Never badmouth competitors by name
